一般按钮
设计 - 将 Ektron eWebEdit400 转换成设计方式。 -
HTML - 将 Ektron eWebEdit400 转换成 HTML 方式。 -
预览 - 将 Ektron eWebEdit400 转换成预览方式。 -
全屏幕 - 将 Ektron eWebEdit400 转换成全屏幕。 -
显示/隐藏边框 - 在编辑区内显示/隐藏表格边框。 -
显示比例 - 改变编辑区内文字显示之大小。 -
管理模组 - 在下拉式选单中选择欲使用的模组或取消使用中的模组。 -
回复工具列位置 - 移动所有工具列至预设位置 -
重复上一次动作 - 重复执行上一次以完成的动作 -
搜寻和取代 - 在编辑区内搜寻(和取代)文字。 Ctrl+F
列印 - 列印 Ektron eWebEdit400 或者个网页的内容。 Ctrl+P
校正 - 拼字检查及校正。 -
剪下 - 剪下被选取的内容并复制到剪贴簿中。 Ctrl+X
复制 - 复制选取的内容至剪贴簿中。 Ctrl+C
贴上 - 将剪贴簿上的内容贴到编辑区中。 Ctrl+V
贴上文字 - 贴上文字内容并移除网页标签。 -
贴上文字(清除字型及大小) - 清除所有文字标签、字型及大小。 -
纯文字贴上 - 仅贴上文字(不包含格式)至编辑区中。 -
贴上 HTML - 贴上 HTML 至编辑区中并保留所有 HTML 标签。 -
复原 - 回覆上一个动作。 Ctrl+Z
取消复原 - 重复上一次复原的动作。 Ctrl+Y
清除格式 - 移除自订格式或所有来自于选取文字上的格式。 -
说明 - 帮助您正确且快速的使用 Ektron eWebEdit400 。 -

插入时间 - 插入正确的时间。 -
管理 Flash - 插入 Flash 档案，并设定其相关属性。 -
管理 Media - 插入 Windows media 物件 (AVI, MPEG, WAV,等)，并设定其相关属性。 -
管理文件 - 插入文件的连结 (PDF, DOC, 等)。 -
插入超连结 - 使所选择的文字或影像具有超连结的功能。 Ctrl+K
移除超连结 - 从所选的文字或影像中移除超连结。 Ctrl+Shift+K
插入特殊符号 - 插入特殊字元 (€ ®, ? ?/font>, 等)。 -
自订连结 - 插入事先定义好之外部或内部连结。 -
管理样版 - 提供事先定义好的 HTML 样版。 -
新增、删除及编辑段落及线段
新段落 - 插入新的段落。 Ctrl+M
段落样式 - 提供标准的文字样式套用在所选择的文字上。 -
文字凸排 - 段落往左凸排。 -
文字缩排 - 段落往右缩排。 -
靠左对齐 - 所选择的段落靠左对齐。 -
置中对齐 - 所选择的段落置中对齐。 -
靠右对齐 - 所选择的段落靠右对齐。 -
齐行 - 将所选择的段落排整齐。 -
项目清单 - 从选择区域中新增项目清单。. -
编号清单 - 从选择区域中新增编号清单。 -
水平线 - 在光标所指的地方插入一条水平线。 -
新增、删除及编辑文字格式
粗体 - 将所选择的文字加入粗体格式。 Ctrl+B
斜体 - 将所选择的文字加入斜体格式。 Ctrl+I
底线 - 将所选择的文字加入底线格式。 Ctrl+U
删除线 - 将所选择的文字加入删除线格式。 -
上标 - 使文字上标。 -
下标 - 使文字下标。 -
字型- 设定文字样式。 -
大小- 设定文字大小。 -
前景颜色 - 改变所选择文字之前景颜色。 -
背景颜色 - 改变所选择文字之背景颜色。 -
CSS 样式 - 将自订或是先定义好的样式套用至所选择的文字中。 -
自订连结 - 插入事先定义好之连结。 -
