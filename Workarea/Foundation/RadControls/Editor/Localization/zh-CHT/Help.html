一般按鈕
設計 - 將 Ektron eWebEdit400 轉換成設計模式。 -
HTML - 將 Ektron eWebEdit400 轉換成 HTML 模式。 -
預覽 - 將 Ektron eWebEdit400 轉換成預覽模式。 -
全螢幕 - 將 Ektron eWebEdit400 轉換成全螢幕。 -
顯示/隱藏邊框 - 在編輯區內顯示/隱藏表格邊框。 -
顯示比例 - 改變編輯區內文字顯示之大小。 -
管理模組 - 在下拉式選單中選擇欲使用的模組或取消使用中的模組。 -
回復工具列位置 - 移動所有工具列至預設位置 -
重複上一次動作 - 重複執行上一次以完成的動作 -
搜尋和取代 - 在編輯區內搜尋(和取代)文字。 Ctrl+F
列印 - 列印 Ektron eWebEdit400 或者個網頁的內容。 Ctrl+P
校正 - 拼字檢查及校正。 -
剪下 - 剪下被選取的內容併複製到剪貼簿中。 Ctrl+X
複製 - 複製選取的內容至剪貼簿中。 Ctrl+C
貼上 - 將剪貼簿上的內容貼到編輯區中。 Ctrl+V
貼上文字 - 貼上文字內容並移除網頁標籤。 -
貼上文字(清除字型及大小) - 清除所有文字標籤、字型及大小。 -
純文字貼上 - 僅貼上文字(不包含格式)至編輯區中。 -
貼上 HTML - 貼上 HTML 至編輯區中並保留所有 HTML 標籤。 -
復原 - 回覆上一個動作。 Ctrl+Z
取消復原 - 重複上一次復原的動作。 Ctrl+Y
清除格式 - 移除自訂格式或所有來自於選取文字上的格式。 -
說明 - 幫助您正確且快速的使用 Ektron eWebEdit400 。 -
關於 Ektron eWebEdit400 - 顯示關於 Ektron eWebEdit400 版本及版權。 -
插入並管理連結、表格、特殊文字、影像及 MEDIA 。
管理影像 - 從上傳之影像檔案中插入影像。 Ctrl+G
設定絕對位置 - 設定一物件的絕對位置(自由移動)。 -
插入表格 - 插入表格至 Ektron eWebEdit400. -
顯示/隱藏邊框 - 顯示/隱藏編輯區內所有表格之邊框。 -
插入範本 - 插入事先設計好的範本。 -
插入表單元素 - 從下拉式選單中插入欲使用的表單元素。 -
插入日期 - 插入正確的日期。 -
插入時間 - 插入正確的時間。 -
管理 Flash - 插入 Flash 檔案，並設定其相關屬性。 -
管理 Media - 插入 Windows media 物件 (AVI, MPEG, WAV,等)，並設定其相關屬性。 -
管理文件 - 插入文件的連結 (PDF, DOC, 等)。 -
插入超連結 - 使所選擇的文字或影像具有超連結的功能。 Ctrl+K
移除超連結 - 從所選的文字或影像中移除超連結。 Ctrl+Shift+K
插入特殊符號 - 插入特殊字元 (€ ®, ? ?/font>, 等)。 -
自訂連結 - 插入事先定義好之外部或內部連結。 -
管理樣版 - 提供事先定義好的 HTML 樣版。 -
新增、刪除及編輯段落及線段
新段落 - 插入新的段落。 Ctrl+M
段落樣式 - 提供標準的文字樣式套用在所選擇的文字上。 -
文字凸排 - 段落往左凸排。 -
文字縮排 - 段落往右縮排。 -
靠左對齊 - 所選擇的段落靠左對齊。 -
置中對齊 - 所選擇的段落置中對齊。 -
靠右對齊 - 所選擇的段落靠右對齊。 -
齊行 - 將所選擇的段落排整齊。 -
項目清單 - 從選擇區域中新增項目清單。. -
編號清單 - 從選擇區域中新增編號清單。 -
水平線 - 在游標所指的地方插入一條水平線。 -
新增、刪除及編輯文字格式
粗體 - 將所選擇的文字加入粗體格式。 Ctrl+B
斜體 - 將所選擇的文字加入斜體格式。 Ctrl+I
底線 - 將所選擇的文字加入底線格式。 Ctrl+U
刪除線 - 將所選擇的文字加入刪除線格式。 -
上標 - 使文字上標。 -
下標 - 使文字下標。 -
字型- 設定文字樣式。 -
大小- 設定文字大小。 -
前景顏色 - 改變所選擇文字之前景顏色。 -
背景顏色 - 改變所選擇文字之背景顏色。 -
CSS 樣式 - 將自訂或是先定義好的樣式套用至所選擇的文字中。 -
自訂連結 - 插入事先定義好之連結。 -
其他鍵盤相關的快速鍵
- 選擇編輯區內所有的文字、影像及表格。 Ctrl+A
- 搜尋網頁中的文字。 Ctrl+F
- 關閉正在使用的視窗。 Ctrl+W
- 關閉正在使用的應用程式。 Ctrl+F4
